Timeline entries

We use different styles of timeline entries depending on the action completed. View documentation on this pattern in confluence (opens in new tab).

DWP pattern for timeline entries (opens in new tab).

We currently have 3 types of timeline entries:

  • claim action
  • agent note
  • NHS note

Claim action is used when an agent completes any action on the claim. For example, when a claim is saved.

Agent note is used when an agent adds a note to the claim, from the claim details page. This is a freetext note.

NHS note is used for a few different NHS related abbreviations. This will not appear currently as we do not accept claims on CRS with NHS treatment, it will only appear when claims have NHS treatment.

Error summary and error messages
